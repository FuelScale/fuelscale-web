Frequently Asked Questions

FAQ - 1

What is Capacitive Fuel Sensing?

Capacitive Level Sensing is based on the fundamental property of capacitor where introduction of any non-conductive material cause change in capacitance of the capacitive sensor with ability for continuous measurement , it can operate as a pure linear function i.e we can sense empty, full or anywhere in between.

Does it work with Petrol Engines?

No, the device only works with diesel engines.

How long does a fuel sensor last?

We give a warranty for one year, but the sensor performs well for more than 3 years in all cases.

Is it just a one time purchase?

We offer the sensor with web application for analysis and monitoring, which comes with the subscription charges.Overall with one time cost of sensor, montly minimum subscription fees for the web application and the sim on gps module is applicable.

Do we need an extra battery for the sensor?

No, there is no need of an extra battery the sensor works well with already installed battery and has minimum power requirements.

Does sensor works with every tank?

We have different callibration process for different tanks and fuel in all kind of rectangular, cylndircal and irregular tanks can be measured with the sensor.

How much time does it take to install one senor?

On an average it take 40-70 mins to install and configure one sensor.

Is the Sensor certified?

Yes, we have all necceasary international certifications (IP67, IK10 etc)
